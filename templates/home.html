Instructions & Information

The objective of this game is to capture disappearing circles with your hands.

All that is required is a webcam

Upon clicking play, the game begins, with the objective of catching as many circles as possible

Goodluck!

Technologies

This application uses MediaPipe with Open-CV, connected to this frontend using a Python Flask backend
